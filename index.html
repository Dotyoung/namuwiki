🌐 사이트 배포하기 (GitHub Pages)

이 HTML 파일을 무료로 인터넷에 올려 친구들과 공유할 수 있어요. 아래 방법을 따라하세요!

  1. GitHub에 가입하고 새 저장소(Repository)를 만드세요.
    이름은 아무거나 상관없어요. (예: kiwi-wiki)
  2. 저장소 페이지에서 Settings → Pages로 이동하세요.
  3. Source를 Deploy from a branch로 선택하고, Branch를 main / /(root)로 설정 후 Save를 누르세요.
  4. 메인 페이지로 돌아와 Add file → Upload files를 클릭하세요.
  5. 이 HTML 파일을 드래그해서 업로드하고, 파일 이름을 index.html로 바꾼 후 Commit changes를 누르세요.
  6. 잠시 후 https://[GitHub아이디].github.io/[저장소이름] 주소로 접속하면 완료!

💡 팁: 파일을 수정하려면 GitHub에서 index.html를 클릭 → 연필 아이콘(✏️) → 수정 → Commit changes 하면 자동으로 반영돼요!
친구들에게 주소만 알려주면 모두가 같은 사이트를 볼 수 있어요.

💡 사용 팁
  • 엔터를 치면 줄바꿈이 됩니다.
  • 문법을 모르겠으면 툴바 버튼을 누르면 자동으로 입력돼요.
  • 색상 코드는 #ff6b6b, #27ae60 같은 형식입니다.
  • 편집 중에는 3초마다 자동 저장되며, "완료" 버튼을 누르면 최종 저장 후 보기 모드로 돌아가요.

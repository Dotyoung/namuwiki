📖 키위나무 사용법

친구들끼리 쉽게 쓰는 인물 위키! 아래 내용을 참고해서 문서를 작성해보세요.

1. 기본 작성법

  • 그냥 타이핑하면 일반 텍스트로 입력돼요.
  • 엔터를 치면 줄바꿈이 됩니다.
  • 위키 문법을 쓰면 다양한 서식을 적용할 수 있어요.

2. 문법 정리 (기호 모음)

**굵게** → 굵은 글씨 *기울임* → 기울어진 글씨 ~~취소선~~ → 취소선 ## 제목 → 큰 제목 ### 소제목 → 작은 제목 > 인용문 → 인용 상자 [[각주:내용]] → 각주(마우스 올리면 보임) [[색상:#ff6b6b]]텍스트[[/색상]] → 색상 입히기 [[이미지:URL]] → 이미지 삽입

3. 각주 사용법

  • 본문 중 원하는 곳에 [[각주:여기에 각주 내용을 쓰세요]] 를 넣으면 됩니다.
  • 미리보기에서 마우스를 올리면(또는 터치하면) 각주 내용이 바로 떠요!
  • 각주 번호는 자동으로 매겨집니다.
💡 팁: 각주 내용에 URL을 넣으면 출처 표시로 활용할 수 있어요!

4. 이미지 삽입

  • 툴바의 "이미지 업로드" 버튼을 눌러 사진을 선택하세요.
  • 또는 직접 [[이미지:이미지URL]] 형식으로 입력할 수도 있어요.
  • 업로드한 이미지는 자동으로 문서에 포함됩니다.

5. 색상 넣기

  • 툴바의 색상 버튼을 누르거나, 직접 색상 코드를 입력하세요.
  • [[색상:#ff6b6b]]빨간 글씨[[/색상]] 형식입니다.
  • 원하는 색상 코드를 마음대로 쓸 수 있어요! (예: #27ae60, #e74c3c 등)

6. 충돌 방지 & 저장

  • 저장할 때마다 히스토리가 자동으로 쌓여요. 실수로 지워도 복구 가능!
  • 히스토리 목록에서 "불러오기"를 누르면 과거 버전으로 돌아갈 수 있어요.
  • 데이터는 브라우저에 저장되며, "내보내기"로 파일로 백업할 수 있어요.
  • 친구들끼리 JSON 파일을 주고받으면서 공유하세요!
💡 팁: 중요한 시점마다 "내보내기"로 백업 파일을 만들어두는 걸 추천해요!

7. 데이터 백업 & 공유

  • 인물 목록 페이지 하단의 "데이터 내보내기" 버튼을 누르면 .json 파일이 다운로드돼요.
  • 이 파일을 친구에게 보내면 친구가 "데이터 불러오기"로 불러올 수 있어요.
  • 여러 사람이 번갈아가며 편집해도, 파일만 주고받으면 충돌 없이 합칠 수 있어요!

8. 자동 백업 & 편집 상태

  • 키보드를 치기 시작하면 "편집 중..." 표시가 떠요.
  • 잠시 멈추면 자동 백업이 실행되고 "자동 백업 완료!" 문구가 뜹니다.
  • 따로 저장 버튼을 누르지 않아도 안심이에요.
